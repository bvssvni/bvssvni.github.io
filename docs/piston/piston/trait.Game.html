Provided Methods

fn render(&self, _context: &Context, _gl: &mut Gl)

Render graphics.

context is a Rust-Graphics context. gl is the Piston OpenGL back-end for Rust-Graphics.

fn update(&mut self, _dt: f64, _asset_store: &mut AssetStore)

Update the physical state of the game.

dt is the delta time from last update in seconds.

fn load(&mut self, _asset_store: &mut AssetStore)

Perform tasks for loading before showing anything.

fn key_press(&mut self, _key: Key, _asset_store: &mut AssetStore)

User pressed a key.

This can be overridden to handle key pressed events.

fn key_release(&mut self, _key: Key, _asset_store: &mut AssetStore)

User released a key.

This can be overridden to handle key released events.

fn mouse_press(&mut self, _button: Button, _asset_store: &mut AssetStore)

Pressed a mouse button.

fn mouse_release(&mut self, _button: Button, _asset_store: &mut AssetStore)

Released a mouse button.

fn mouse_move(&mut self, _x: f64, _y: f64, _asset_store: &mut AssetStore)

Moved mouse cursor.

fn mouse_relative_move(&mut self, _dx: f64, _dy: f64, _asset_store: &mut AssetStore)

Moved mouse relative, not bounded by cursor.

fn viewport<W: GameWindow>(&self, game_window: &W)

Sets up viewport.

A viewport is the region of the window where graphics is rendered.

fn should_close<W: GameWindow>(&self, game_window: &W) -> bool

Whether the window should be closed.

When this is true the application shuts down. This can be overridden to emulate a user closing the window. One can also override this method to prevent window from closing.

fn swap_buffers<W: GameWindow>(&self, game_window: &W)

Swaps the front buffer with the back buffer.

When called, This shows the next frame. The graphics is rendered to the back buffer. The front buffer is displayed on the screen.

fn handle_events<W: GameWindow>(&mut self, game_window: &mut W, asset_store: &mut AssetStore)

Handles events using current game window settings.

This can be overriden to do custom event handling.

fn run<W: GameWindow>(&mut self, game_window: &mut W, asset_store: &mut AssetStore)

Executes a game loop.

The loop continues until should_close returns true.
